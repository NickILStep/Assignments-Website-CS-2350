Activities:

Module 1:

1-1 Curbside Thai
In this assignment we learned how to write HTML in a variety of ways using sectioning tags and other common HTML tags.
1-3 Tri and Succeed Sports
In this assignment we learned more about sectioning off different parts of the webpage, and we looked at more detailed ways to customize lists and paragraphs using CSS.

Module 2:

2-1 Pandaisia Chocolates
In this assignment we learned how to lay out a website using different positioning methods.
2-3 Dakota Listener Radio
In this assignment we learned how to set up and style tables including borders and column and row spans.

Module 3:

3-1 Komatsu Family
In this assignment we learned how to do some graphic design using both html and css. We learned how to add and style borders and images, as well as add 3d effects. We also used an image map.
3-4 Red Ball Pizza
In this assignment we learned how to create a web form using all kinds of different input methods, as well as styling those inputs and running them through client-side validation.

Module 5:

5-1 Tulsa New Year
This was our first assignment with JavaScript. We learned many of the basics of coding in this language, but the main focus of this assignment was creating a website with a continuously updating date and time as well as a message pop-up.
5-3 Lyman Hall Theater
In this assignment we learned more about JavaScript. We learned how to use arrays, loops, if-statements, comparison operators, and more to add functionality to a calendar on a webpage.

Module 6:

6-1 JPF Hanjie Puzzle
In this assignment we used JavaScript to create an interactive puzzle game that used for-loops to alter background color, as well as changing the cursor and checking for the completed solution.
6-3 Bridger College
In this assignment we learned how to create html tags and add atributes to them with JavaScript, as well as creating and disabling/enabling embedded stylesheets.
6-6 Coctura Home Kitchen
In this assignment we learned how to send form information between pages, as well as decrypting this information and validating form submissions even further.
